# شركة الماهر للصيانة والخدمات - الموقع التعريفي والبروفايل الرقمي

موقع إلكتروني تفاعلي متكامل يمثل البروفايل الرقمي لشركة **الماهر للصيانة والخدمات** (المتخصصة في صيانة الأجهزة المنزلية وأنظمة التكييف بمكة المكرمة منذ عام 2000). تم تصميم الموقع بأسلوب عصري يعتمد على واجهات لوحات التحكم (Dashboards) لاستعراض خدمات الشركة، مؤشرات الأداء، والشراكات المعتمدة.

## 🛠 التقنيات المستخدمة (Tech Stack)

* **HTML5 & CSS3:** البنية الأساسية والتنسيقات المخصصة.
* **Tailwind CSS:** بناء واجهة مستخدم متجاوبة وسريعة الاعتماد على نظام (Utility-first).
* **Chart.js:** توليد رسوم بيانية تفاعلية (دائرية وشريطية) لاستعراض توزيع الخدمات والماركات المدعومة.
* **Font Awesome:** نظام الأيقونات الرقمية للأقسام ووسائل التواصل.
* **Google Fonts:** استخدام خط (Cairo) المعتمد للمحتوى العربي المهني.
* **Vanilla JavaScript:** إدارة التنقل بين الأقسام (Single Page Application) والتحكم في القوائم الجانبية للهواتف.

## ✨ الميزات الرئيسية (Key Features)

* **تفاعلية كاملة (SPA):** التنقل بين الأقسام بسلاسة دون إعادة تحميل الصفحة.
* **لوحة إحصائيات متقدمة:** استعراض مؤشرات الأداء الرئيسية (معدل نجاح إصلاح 98%، دعم 24/7).
* **واجهة عصرية (UI/UX):** تطبيق تأثيرات زجاجية (Glassmorphism) وتوهج نيوني (Neon Glow) مع خلفيات تقنية مخصصة (Circuit Pattern).
* **توافق مطلق (Responsive Design):** شريط تنقل جانبي ذكي يتكيف تلقائيًا مع الشاشات المكتبية والهواتف المحمولة.
* **دعم الماركات العالمية:** استعراض توثيقي لأكثر من 12 علامة تجارية معتمدة.

## 📂 أقسام الموقع (Sections)

1.  **نبذة عن الشركة:** التاريخ، الرؤية، والمزايا التنافسية.
2.  **الخدمات والتخصصات:** تفاصيل صيانة (الغسالات، غسالات الصحون، المجففات، التكييف، الأفران، المكانس).
3.  **معلومات التواصل:** قنوات الاتصال المباشر والدعم الفني المتاح على مدار الساعة (24/7).
4.  **الموقع الجغرافي:** نطاق التغطية الجغرافية في منطقة مكة المكرمة.
5.  **تقييمات العملاء:** مؤشرات رضا العملاء وسرعة الاستجابة والاحترافية.
6.  **الشهادات والشراكات:** توثيق الوكالات المعتمدة (سامسونج، إل جي، بوش، أريستون، إلخ).
7.  **الإحصائيات والتحليلات:** مقارنات رقمية ومخططات بيانية لأداء الخدمات.
